# 아래 mkdown을 참조해서, 자신의 github에 사용하자
- username=[사용자이름]
- langs_count=[화면에 표기될 언어갯수]
- layout=[화면에 리스트 형식 혹은 통계치로 보여질지 여부]
- theme=[default는 white배경. 그 외 theme는 github-readme-stats 참조]
- ()﻿ = [위에 만든 자신의 username로 생성된 Repo주소]

example : 
﻿[![Top Langs](https://github-readme-stats.vercel.app/api/top-langs/?username=benny&langs_count=10&layout=compact&theme=dark)](https://github.com/Wonsang222/Wonsang222)
